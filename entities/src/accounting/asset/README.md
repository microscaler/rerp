# Asset Management Service

Entities for managing fixed assets, depreciation, and asset transactions.

## Planned Entities

- **Asset** - Fixed assets (equipment, property, etc.)
- **AssetCategory** - Asset categorization
- **AssetDepreciation** - Depreciation schedules and calculations
- **AssetTransaction** - Asset purchase, sale, disposal transactions
