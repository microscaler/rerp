# Getting Started

## What is DocuPipe?   [Skip link to What is DocuPipe?](https://docs.docupipe.ai/docs/getting-started\#what-is-docupipe)

DocuPipe is a document AI platform that lets you define data-extraction workflows and run them with unrivaled reliability. With DocuPipe you can:

1. **Parsing** millions of documents and scans, in any format.
2. **Define** the specific information you need to extract. We call that definition a **Schema**.
3. **Extract** information reliably by applying your schema to each document. This handles variable layouts, formats, and even 100+ languages while returning a consistent set of fields called a **Standardization**.
4. **Build** [Workflows](doc:workflow-split-classify-extract) that combine Classification, **split** operations, and more to orchestrate complex extraction tasks.
