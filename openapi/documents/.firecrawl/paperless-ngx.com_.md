Paperless-ngx is an open-source document management system that enables users to convert and manage their physical documents in a searchable digital archive. Key features include organized indexing with tags, local data storage, optical character recognition (OCR) supporting over 100 languages, and a customizable web application interface. Users can search documents using full-text search with auto-completion, import documents via email, and manage multi-user permissions. The system is the successor to previous projects Paperless and Paperless-ng, and it supports various scanners and software through a community-driven approach.
